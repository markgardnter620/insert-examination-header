InsertExaminationHeader (IEH)
A professional-grade Word macro suite designed to automate the placement of examination headers in transcripts of legal proceedings

Overview
IEH replaces manual header formatting with a robust, automated workflow. It handles section isolation and header text alignment allowing you to process transcripts with precision and speed.

Key Features
♦ Automated Workflow: Insert headers across a range of pages in under 10 seconds.
♦ Configurable: Choose between Title Case or ALL CAPS, and Center- or Left-Aligned positioning.
♦ "No-Nonsense" Engineering: Built for stability and speed, with Undo support and clear error handling.
♦ Complete Suite: Includes tools for shortcut assignment, clearing existing headers, and configuration management.

Quick Start
♦ Download: Obtain the latest release from the Releases page.
♦ Install: Extract the .zip file. In Word, press Alt+F11, import InsertExaminationHeader.bas, and Compile the project.
♦ Run: Use your assigned shortcut (default: Shift+Ctrl+Alt+H) to trigger the macro on any sloppily selected range.

For full installation details and instructions, refer to the included README_INSERT_EXAMINATION_HEADER.pdf.
